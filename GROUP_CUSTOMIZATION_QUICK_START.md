# Group Customization - Quick Start

## What You Can Do Now

Your data grid now supports **customizing individual groups** with colors, labels, and notes!

## How to Use (3 Simple Steps)

### Step 1: Group Your Data
Click the "Group By" button in the toolbar and select a column:

```
[Group By ▼] → Select "Department" ✓
```

Your data will be grouped:
```
▶ Engineering (15 users)
▶ Sales (8 users)
▶ Marketing (12 users)
```

### Step 2: Customize a Group
Hover over any group header and click the menu icon (⋮):

```
▶ Engineering (15 users)  [⋮]
                           ↓
                    [Customize Group]
```

### Step 3: Apply Your Customization
In the dialog:
- **Pick a color**: Choose from color picker or enter hex code
- **Add a label**: Enter custom text (e.g., "Core Team")
- **Add notes**: Add metadata or descriptions

Click **Save** and you're done!

## Examples

### Color-Code Teams
```
▶ Engineering (15 users)     [Blue background]
▶ Sales (8 users)            [Green background]
▶ Marketing (12 users)       [Purple background]
```

### Add Custom Labels
```
▶ active → "✓ Active Users" (50 users)
▶ inactive → "⏸ Inactive" (10 users)
▶ pending → "⏳ Pending Approval" (5 users)
```

### Add Important Notes
```
▶ high priority (20 items)
  Note: "Requires immediate attention - escalate to manager"
```

## Features

✅ **Persistent** - Customizations survive page refreshes
✅ **Dataset Agnostic** - Works with any data (users, products, orders, etc.)
✅ **Easy to Use** - Just 3 clicks to customize
✅ **Easy to Reset** - Click "Reset" to remove customization

## That's It!

No configuration needed. Just start grouping and customizing!
